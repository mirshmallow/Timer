# ⏳ Python Timer

A simple countdown timer built with Python that supports hours, minutes, and seconds.

## 🚀 Features
- Input time in hours, minutes, or seconds
- Countdown display (HH:MM:SS)
- Input validation
- Option to add time before starting

## 🧠 How it works
The program converts all input into seconds and then runs a countdown loop, updating the time every second.

## ▶️ How to run
```bash
python timer.py
